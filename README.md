# Scenes-Classification-transfer-learning
transfer learning from [places-365-vgg16 CNN](https://github.com/GKalliatakis/Keras-Application-Zoo/blob/master/vgg16_places_365.py) trained on [MIT Places 2 Dataset](http://places2.csail.mit.edu/) on 365 scene class to smaller amount of scene classes or custom scene classes.

* install requirement using pip3
```sh
$ pip3 install -r requirment.txt
```

* [SceneClassification.ipynb](https://github.com/OmarSayedMostafa/Scenes-Classification-transfer-learning/blob/master/SceneClassification.ipynb) jupyter notebook which contains full instruction and details rquired to understand what's going on.
